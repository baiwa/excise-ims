ผู้ประกอบการที่ชำระภาษีครบ 24 เดือน

อัตราการชำระภาษีคงที่ (+-ไม่เกินร้อยละ 5)

ผู้ประกอบการที่ชำระภาษีไม่ครบ 24 เดือน

อัตราการชำระภาษีเพิ่มขึ้นไม่เกินร้อยละ 5
